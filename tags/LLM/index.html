CS336 Scaling laws

lecture9-Scaling laws basic

  1. 缩放定律意义是为语言模型的行为建立简单的预测规律；核心思想是同从小模型入手将他们放大，并借此改良工程方法。

  2. 缩放定律的背景是只要投入可预测资源，就能获得可预测能力的提升，即缩放和收敛是自然的，大数据量带来的收敛是自然的。

  3. 缩放是多方面的，算力、参数量、数据量都是由缩放规律的，其中参数量、数据量遵循幂律关系。

  4. 这其中开头和结尾的平滑曲线符合幂律分布。
